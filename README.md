A display of various simulator stats for OSgrid
Please note, that some of the functions used require OSSL. The commands are:
  -osGetMapTexture()
  -osLoadedCreationDate()
  -osLoadedCreationTime()
  -osLoadedCreationID()
  -osGetRegionStats()
  -osGetSimulatorVersion()
  -osGetGridName()
